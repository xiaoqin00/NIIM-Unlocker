### 方法一

直接打开下载好的证书文件，选择“证书安装程序”，选择 **CA 证书**，为证书输入一个名称，点按**确定**。

![](https://img02-xusong.taihe.com/133901_16801891180x28e735a90_[1018_567_932].png)

### 方法二

1. 打开手机的“设置”应用。
2. 搜索**加密与凭据**、**证书**，找到**从SD卡安装**或**从存储设备安装**。

   ![](https://img02-xusong.taihe.com/118686_16801892010x2873675a0_[1920_1080_1084].png)

   ![](https://img02-xusong.taihe.com/140256_16801892240x28c208a10_[1920_1080_1632].png)
3. 依次点按**安装证书** >**CA 证书**。
4. 找到证书所在的位置
   - 通常位于 **手机存储/Download/Browser** 目录中，文件名为**FiddlerRoot**
   
     ![](https://img02-xusong.taihe.com/133814_16801892920x28ef4b4e0_[634_401_404].png)
   
     ![](https://img02-xusong.taihe.com/133175_16801893260x28a0e3770_[820_584_634].png)
5. 点按相应文件。
   - 如需要输入密码。点按**确定**。
6. 为证书输入一个名称。

   ![](https://img02-xusong.taihe.com/133901_16801891180x28e735a90_[1018_567_932].png)
7. 点按**确定**。

**提示**：如果您还没有为手机设置 PIN 码、图案或密码，系统会提示您进行设置。

## 开启解锁功能

打开 Clash for Android 应用程序。点击**点此启动**。

以后每次使用前只需要重复这一步即可。

![](https://img02-xusong.taihe.com/119844_16801895860x28d75c430_[1096_530_752].png)

![](https://img02-xusong.taihe.com/129551_16801896000x28837cc20_[1060_669_966].png)

## 最后一步

打开精臣云打印APP，显示**"欢迎使用云打印解锁版"**。

**登录账号**并**连接设备**后，点击右下角“我的”，将**帐号UID**和**设备ID**复制给工作人员，后台授权后即可解锁成功。
