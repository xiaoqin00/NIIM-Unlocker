### 方法一

直接打开下载好的文件，选择“证书安装程序”，选择 **VPN 证书**，为证书输入一个名称，点按**确定**。

### 方法二

1. 打开手机的“设置”应用。
2. 搜索**证书**。
3. 依次点按**安装证书** ![然后](https://lh3.googleusercontent.com/WD3LKKej34vq3cZXwilgeahIPOiokN2uarmkDxtMqKMFg4SSys8BkOBJbn4_4R930gE=w36-h36) **VPN 证书**。
4. 找到证书所在的位置。
5. 点按相应文件。
   - 如需要输入密码。点按**确定**。
6. 为证书输入一个名称。
7. 点按**确定**。

**提示**：如果您还没有为手机设置 PIN 码、图案或密码，系统会提示您进行设置。

## 开启解锁功能

打开 Clash for Android 应用程序。点击**点此启动**。

以后每次使用前只需要重复这一步即可。

![](https://img01-xusong.91q.com/73B42CDB-5FEE-48CE-8535-B2C250DAD125.jpg)

## 最后一步

打开云打印APP，查看会员到期时间，若显示2099-12-31则为解锁成功，连接打印机，测试效果。

Tips：若解锁不成功，请尝试清除云打印APP数据后重新打开。
